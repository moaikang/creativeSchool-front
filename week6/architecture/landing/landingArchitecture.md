# 랜딩 페이지 설계

### Project Directory Structure   
- components: 컴포넌트 파일들이 위치하는 폴더.   
- containers: 컨테이너 파일들이 위치하는 폴더. 주로 state를 props로 매핑하는 코드들이 위치.   
- pages: Routing을 위한 페이지 파일들이 위치하는 폴더.   
- modules: redux 작업을 위한 폴더.    
- lib : api 호출을 위한 코드를 모아두는 폴더.
